a clean slate: deleting all tweets with python

As my first order of business, I was looking to to delete all prior tweets from my account, since I plan to start using Twitter in a more professional capacity from now on. To do so, I see there are various sites and services out there to assist with that. But part of beginning this blog was a commitment to exploring how I can use official APIs, third-party libraries or my own code from scratch to accomplish things on the web.

So I found this nice Python script, courtesy of Dave Jeffery.

I registered my twitter application under bird_eater (ha, get it? because it’s Python getting rid of tweets… eh) and put in my key and secret. I then followed the directions from the script’s output et voilà! I am now tweet-free. I can restart my Twitter profile with a clean slate — notwithstanding the other parts of my account.

In fact, why not take care of those things, too? To take this further, I should expand on this script to make it a bit more sophisticated. Some ideas could be deleting certain tweets, unfollowing accounts, removing likes and lists, etc. That could be a promising project.
